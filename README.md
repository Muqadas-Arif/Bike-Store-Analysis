**A complete end-to-end data analysis project** using Microsoft SQL Server, Python (Pandas, Matplotlib, Seaborn), and real-world retail bike store data.

### Project Overview
This project analyzes sales data from a fictional bike store chain (BikeStores database).  
Goal: Understand sales trends, identify top performers, diagnose performance drivers, and provide actionable business recommendations.

**Key Questions Answered:**
- What are the overall sales trends and peak months?
- Which stores, products, and categories drive the most revenue?
- Why do sales peak in April and crash in May?
- How does discount strategy impact revenue?

### Tech Stack
- **Database**: Microsoft SQL Server (SSMS)
- **Languages**: SQL (queries, joins, aggregations), Python 3.11
- **Libraries**: pandas, matplotlib, seaborn, sqlalchemy, pyodbc
- **Tools**: VS Code + Jupyter Notebook, SSMS
